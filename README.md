NAME
====

Text::UpsideDown - Flip text upside-down using Unicode

SYNOPSIS
========

    use Text::UpsideDown;
    say upside_down("foo");
    # prints "ɟoo"

DESCRIPTION
===========

This module will flip text upside-down using Unicode.
